# DGA Registry

A collection of Saudi Data Governance UI components and utilities for Next.js apps. Built on **shadcn/ui** with DGA design tokens, RTL support, and IBM Plex Sans Arabic typography.

---

## New Project Setup

### Next.js

Run the `init` command to create a new Next.js project or set up an existing one.
The `--radix` flag is required to use DGA Registry components.

<InstallCommand variant="init-next" />

Need RTL (Arabic) support? Add the `--rtl` flag:

<InstallCommand variant="init-next" rtl />

### Laravel

<InstallCommand variant="init-laravel" />

With RTL:

<InstallCommand variant="init-laravel" rtl />

---

## Add the DGA Registry

This will enable short-hand `@dga/` imports in your `components.json`:

<RegistryBlock />

## Add the Design Tokens

Once initialized, add this import to the top of your `globals.css`:

<InstallCommand variant="globals" />

This loads all DGA color tokens, spacing, and typography variables.
