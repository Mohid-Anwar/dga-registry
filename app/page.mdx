# DGA Registry

A collection of Saudi Data Governance UI components and utilities for Next.js apps. Built on **shadcn/ui** with DGA design tokens, RTL support, and IBM Plex Sans Arabic typography.

## Installation

Add any component to your project with a single command:

<InstallCommand componentName="[component]" />

## Getting Started

1. Install the shadcn CLI if you haven't already:
   <InstallCommand variant="init" />

1.1 Quick Configuration (Optional **but recommended**)

Add registries to your components.json:

```json
{
  "registries": [
    {
      "name": "DGA Registry",
      "url": "https://dga-registry.vercel.app/r"
    }
  ]
}
```
   
2. Run the install command for any component you need.

3. Import and use the component in your app.
